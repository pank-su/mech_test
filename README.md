# Тренажер по механике

Открой `index.html` в браузере. Вопросы лежат в `data/questions.js`, оригинальные страницы PDF - в `assets/pdf-pages`, выделенные схемы и формулы - в `assets/question-media`.

## Как добавить правильный ответ

В `data/questions.js` у вопроса поле `correct` хранит массив индексов вариантов, начиная с нуля.

Пример для одного ответа:

```js
correct: [2]
```

Пример для нескольких ответов:

```js
correct: [0, 3]
```

## Как добавить новый вопрос

Добавь объект в массив `questions` нужного теста:

```js
{
  id: "test1-016",
  source: "свой источник",
  page: 16,
  type: "single",
  text: "Текст вопроса",
  options: ["Вариант 1", "Вариант 2"],
  correct: [0],
  pageImage: null
}
```
